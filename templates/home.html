Malicious Node Detection System

A Hybrid Machine Learning Approach for High Mobility Vehicular Networks

Project Overview

This system implements advanced machine learning algorithms to detect malicious nodes in high mobility vehicular networks. It uses multiple classification models to identify various types of network attacks including denial of service, Sybil attacks, and blackhole attacks.

Classification Targets:
  • denial_of_service: Low, Medium, High, Very High
  • sybil_attack_attempts: None, Low, Medium, High, Very High
  • blackhole_attack_attempts: None, Low, Medium, High, Very High
ML Algorithms:
  • K-Nearest Neighbors (KNN)
  • Support Vector Classifier (SVC)
  • Naive Bayes Classifier
  • Stacked Tao Tree Classifier
